ThermoWave · Version 2.3 · EN ISO 13786

Summer Thermal Behaviour of Constructions

Dynamic thermal parameters according to EN ISO 13786, based on the calibrated Excel matrix method.

1. Load, name and define construction

Workflow: Load a saved construction or build a new one → add layers → calculate → save construction → add it to comparison → clear input → repeat for up to 10 constructions.

2. Add layers from inside to outside

Select a material from the database or enter your own values. Decimal point is used for display and export; decimal comma is still accepted when typing.
