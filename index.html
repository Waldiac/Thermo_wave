Manual / Anleitung

Schnellstart

  1. Geben Sie einen Namen für die Konstruktion ein oder laden Sie eine gespeicherte Konstruktion.
  2. Fügen Sie die Schichten von innen nach außen hinzu. Sie können Materialien aus der Datenbank übernehmen oder eigene Werte eingeben.
  3. Prüfen Sie die Ergebnisse der aktuellen Konstruktion.
  4. Speichern Sie die Konstruktion und übernehmen Sie sie in den Vergleich.
  5. Leeren Sie die Eingabe und erstellen oder laden Sie die nächste Konstruktion.
  6. Nutzen Sie Vergleichstabelle, Vergleichsgrafik und Bericht zur Einordnung.

Empfohlener Workflow

Das Tool ist so gedacht, dass eine Konstruktion immer zuerst aufgebaut oder geladen wird. Danach kann sie gespeichert und zusätzlich in den Vergleich übernommen werden.

  • Konstruktion speichern legt sie in der lokalen Browser-Bibliothek ab.
  • Zum Vergleich hinzufügen/aktualisieren übernimmt die Konstruktion in den aktuellen Vergleichssatz.
  • Eingabe leeren startet eine neue Konstruktion, ohne den Vergleich zu löschen.
  • Der Vergleich ist auf bis zu 10 Konstruktionen begrenzt.

Die Schichtreihenfolge ist immer von innen nach außen. Über die Pfeile in der Schichttabelle können Schichten nach oben oder unten verschoben werden.

Kennwerte verstehen

U₀

Stationärer Wärmedurchgangskoeffizient. Er beschreibt den klassischen U-Wert für stationäre Bedingungen.

|Y₁₂|

Periodischer Wärmedurchgangskoeffizient. Anschaulich ist dies ein dynamischer U-Wert für die tägliche Temperaturwelle. Beispiel: |Y₁₂| · 12 K ergibt die dynamische Wärmestromamplitude in W/m².

f

Dekrementfaktor. Je kleiner f ist, desto stärker wird die äußere Temperaturwelle durch den Aufbau gedämpft.

1/f

Amplitudendämpfung. Je größer 1/f ist, desto stärker ist die anschauliche Dämpfung.

Delay h

Praktische Verzögerung der periodischen Wärmewelle. Größere Werte können günstig sein, wenn die verzögerte Wärme abends oder nachts abgeführt werden kann.

χ₁

Raumseitig wirksame Wärmespeicherfähigkeit. Sie ist besonders wichtig bei solaren Einträgen, hohen Glasanteilen oder internen Lasten.

χ₂

Außenseitig wirksame Wärmespeicherfähigkeit. Sie beschreibt die dynamische Speicherwirkung an der Außenseite.

Der kombinierte Dynamikwert ist keine Bewertungsklasse nach EN ISO 13786. Er ist eine transparente Lesehilfe. Je nach Bewertungsmodus werden Außenwärmewelle, ausgewogene Betrachtung oder solare/interne Lasten unterschiedlich gewichtet.

Daten speichern, laden und sichern

  • Gespeicherte Konstruktionen werden lokal im Browser abgelegt.
  • Über Projekt-JSON exportieren sichern Sie aktuelle Konstruktion, Bibliothek und Vergleichsauswahl in einer Datei.
  • Über Projekt-JSON importieren können diese Daten später wieder geladen werden.
  • Eigene Materialien können über Material-JSON exportieren/importieren separat gesichert oder ausgetauscht werden.
  • Browserwechsel, anderer PC, Cache-Löschung oder privater Modus können lokale Daten unzugänglich machen. Deshalb wichtige Projekte immer als JSON exportieren.
